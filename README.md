## 💼 Business Pitch
This website was built to help **RJ Cafe** grow its online presence. Many local cafés lose potential customers simply because they have no website. This project solves that by providing:

- A **professional online identity** for the café
- A **digital menu** so customers can browse before visiting
- A **contact/order form** to capture leads directly
- A **mobile-friendly design** since most users browse on phones

This website can help RJ Cafe attract more customers, increase brand trust, and eventually enable online ordering.
